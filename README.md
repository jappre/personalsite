# Geek Engineer Pixel Portfolio

A minimalist, retro-themed personal website for geek engineers.
Built with raw HTML, CSS, and vanilla JavaScript. No frameworks, just pixels.

## Features

- **Retro Aesthetic**: CRT scanline overlay, pixel fonts (Press Start 2P, VT323), and custom pixel cursors.
- **Interactive Terminal**: Contact form styled as a command-line interface.
- **Theme Toggle**: Switch between "Cyberpunk" (Dark) and "Retro" (Light) modes.
- **Easter Eggs**: Try the Konami Code (`↑ ↑ ↓ ↓ ← → ← → B A`) to unlock a secret achievement.
- **Typing Effect**: Dynamic text animation in the hero section.

## How to Run

Simply open `index.html` in any modern web browser.

```bash
# MacOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

## Customization

- Edit `index.html` to change content.
- Modify `css/style.css` to tweak colors (variables at the top).
- Update `js/main.js` to add more easter eggs or change the typing text.
